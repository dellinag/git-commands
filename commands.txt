Commonly used commands in git:

  How do we bring a repo down to out local machine:
git clone https://github.com/dellinag/portal-134.git

  How do we see how things are going in git:
git status

  How do we stage all our files at once:
git add . 

  How do we bring all of the staged filed into the repo:
git commit -m "add present tense message/adds files to this branch of repo"

  How do we send our files to the server:
git push
